Welcome to Endfire

We provide full-stack consulting, contracting, and education. We are the engineering team behind Truprice.Healthcare, and when we aren't busy developing innovative healthcare software, we conjure web apps that are fire.

Full-stack learn sprint

Our time is valuable, your time is valuable, so let's stop wasting everyone's time. In just 16 hours, we provide the know-how, tools, and resources to master every layer of the web stack. From the basics of Javascript to test-driven-development, you'll walk away with an understanding of the latest and best web development technologies.

But wait, there's more. After the 16 hours is over, we'll help you build your very own web app. Show it off to your prospective colleges, jobs, friends, dog, whoever.

Curriculum

We teach a lot in this class within 16 hours on purpose. The goal isn't to master any one piece of the web stack, but to become exposed to the latest web technologies and to gain an understaning of how to move forward in this field.

  • Intro to Javascript
  • HTML5 & CSS3 & CSS languages like LESS & SASS
  • Using power text editors like Sublime, Atom, Vim
  • Cloud services like Divshot, Heroku, AWS, Google
  • Building RESTful APIs and other API standards
  • Node.js, Express, Lodash, Restle
  • Build tools like Gulp, Grunt, NPM
  • Designing the frontend with MVC frameworks
  • Designing NoSQL vs. SQL databases
  • Version control with git and GitHub

We hand select books and other resources that will help you the most, and we offer our own team as a learning resource even after the 16 hours are over.
